Shadows

Shadows are made by: box-shadow

Shadows are usually a darker shade of their base color: You can use rgba where the a determines the intensity of the color.

It takes 4 arguments:

  1. horizontal position
  2. vertical position
  3. blur

  4. color

    box-shadow: 5px -5px 10px rgba(0, 0, 0, 0.15);
